Floating Point Systolic Array

This project focuses on the design and simulation of an 8x8 systolic array for matrix multiplication using 32-bit floating point numbers.

  • Designed a systolic array using Verilog for matrix multiplication.
  • Implemented floating-point adder, multiplier, and processing element (PE) block.
  • Developed 4x4 and 8x8 systolic arrays to enhance performance of Deep Learning Algorithms.
  • Simulated using Xilinx Vivado to verify design functionality.
